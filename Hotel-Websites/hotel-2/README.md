# hotel-2

A ready-to-deploy template in the Hotel-Websites category.

## Features
- Responsive design (mobile-first)
- Modern CSS with custom properties
- Vanilla JavaScript (no dependencies)
- Cross-browser compatible

## Customization
Edit `index.html` for content, `style.css` for styling, and `script.js` for behavior.

## Deploy
```bash
../../deploy deploy Hotel-Websites/hotel-2 ~/my-project
```
