# InvestPro - Portfolio Tracker

An investment platform template with:
- Portfolio summary cards (total value, gain/loss, dividends, cash)
- Holdings table with share/price/value/change columns
- Market watchlist with real-time style prices
- Interactive stock chart (Canvas-based line chart)
- Responsive sidebar layout

## Customization
- Edit holdings/watchlist data arrays in `script.js`
- Chart data series can be replaced with real API data
- CSS variables in `:root` control theme
