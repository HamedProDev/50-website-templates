# FinVault - Banking Dashboard

A modern fintech/banking dashboard template with:
- Account overview cards (checking, savings, investment, credit)
- Transaction history table with status badges
- Money transfer form with validation
- Spending breakdown bar chart with category labels
- Responsive mobile-first layout with sidebar navigation

## Customization
- Edit CSS variables in `:root` for colors, spacing, radius
- Replace transaction data in `script.js`
- Account cards can be duplicated in the HTML grid
