# NovaEx - Crypto Exchange

A dark-themed crypto exchange dashboard with:
- Live price chart with time period selectors (Canvas)
- Buy/sell order form with real-time total calculation
- Wallet balance cards with multiple cryptocurrencies
- Transaction history feed
- Base currency selector
- Market ticker header

## Customization
- Update wallet balances and transaction data in `script.js`
- Chart data series is simulated; replace with API data
- Edit `:root` CSS variables for theme colors
