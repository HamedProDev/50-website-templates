# ArtBlock - NFT Marketplace

A modern NFT marketplace template featuring:
- NFT gallery grid with gradient art placeholders
- Bid badges and auction countdown timers
- Collection stats bar (volume, listings, creators)
- Top creators section with follow/unfollow
- Responsive layout with sticky header

## Customization
- Replace NFT/creator data arrays in `script.js`
- NFT gradients can be customized per card
- CSS variables in `:root` control the theme
