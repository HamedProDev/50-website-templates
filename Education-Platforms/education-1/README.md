# SkillForge - Online Learning Platform

Online learning template with course catalog grid, instructor profile cards, student dashboard with progress bars, enrollment system with filter tabs, and pricing plans.

## Features
- Course catalog with category filtering (Web, Data Science, Design, Mobile)
- Instructor profile cards with ratings
- Student dashboard with progress bars and deadlines
- Enrollment toggle (click to enroll/unenroll)
- Responsive mobile-first design
- CSS custom properties for easy theming
