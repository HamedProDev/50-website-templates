# Hero Kit — 5 Hero Layouts

A showcase of 5 distinct hero section layouts in a single page with tab-based switching.

## Layouts
1. **Centered** — Classic centered hero with chip badge
2. **Split** — Text + mock chart visual side by side
3. **Fullscreen BG** — Gradient background with overlays
4. **Left-Aligned** — Feature list with left alignment
5. **Minimal** — Clean, distraction-free minimal hero

## Features
- Tab-based layout switching
- Smooth fade-in transitions
- Fully responsive
- Ready-to-use components
