# aiagent-1

A ready-to-deploy template in the AI-Agents category.

## Features
- Responsive design (mobile-first)
- Modern CSS with custom properties
- Vanilla JavaScript (no dependencies)
- Cross-browser compatible

## Customization
Edit `index.html` for content, `style.css` for styling, and `script.js` for behavior.

## Deploy
```bash
../../deploy deploy AI-Agents/aiagent-1 ~/my-project
```
